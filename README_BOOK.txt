
# Livro em AR (texturas externas)
O livro usa `models/book_open.gltf` (JSON) + `book_open.bin` e **PNG externos**:
- Substitui `book_left_page.png` e `book_right_page.png` pelas tuas imagens (mesmos nomes).
- O material das páginas é **UNLIT** (KHR_materials_unlit) e **doubleSided** para evitar escurecer.
- Colocação: `ar-placement="wall"` automaticamente quando usas `?planet=livro`.
